# Setup Guide

## Installation

Install dependencies:

```bash
npm install
```

## Configuration

Create a `.env` file with the following:

```
DATABASE_URL=postgresql://localhost/mydb
JWT_SECRET=your-secret-key
```

## Running the App

Start the development server:

```bash
npm run dev
```

## Testing

Run tests using `validateOrderInput()` utility function.
